Beschreibung zum Anlegen einer MySQL Datenbank mit Namen erica und Zugriff mit User wrzl und Passwort prmft:

  1. $ mysql -uroot -p mysql
  2. Datenbank anlegen:
    mysql> CREATE DATABASE erica;
  3. User anlegen:
    mysql> INSERT IGNORE INTO user (Host, User, Password) VALUES ('localhost','wrzl', password('prmft'));
    mysql> INSERT IGNORE INTO db (Host, Db, User, Select_Priv, Insert_Priv, Update_Priv, Delete_Priv, Lock_tables_priv) VALUES ('localhost','erica','wrzl','Y','Y','Y','Y','Y');
    mysql> FLUSH PRIVILEGES;

Beschreibung zum Hinterlegen des DB-Host, dem Namen der DB, des zugreifenden Users und dessen Passwortes:

  1. Im Modul connect.prj den Namen des Projektes setzen (kann z.B. der Name des lokalen Users sein):
    $_E['projekt']="erica";
  2. In der Konfigurationsdatei /etc/apache2/conf.d/httpd.conf die Variablen Host DB user pw setzen, dabei im Variablennamen mit dem Namen des Projektes beginnen:
    SetEnv ericaHost "localhost"
    SetEnv ericaDB "erica"
    SetEnv ericaDBuser "wrzl"
    SetEnv ericaDBpw "prmft"
